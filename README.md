🎓 Career-recommendation-system
A machine learning based Career Recommendation System using Streamlit that suggests suitable careers based on user interests.

This project is a Machine Learning-based Career Recommendation System that suggests suitable career paths based on user interests and skills.

🚀 Features
- 40+ Yes/No based questions
- Multiple ML models (Random Forest, KNN, SVM, Decision Tree)
- Ensemble Learning (Voting Classifier)
- Top 3 career predictions with confidence score
- Interactive UI using Streamlit

---

🧠 Technologies Used
- Python
- Streamlit
- Scikit-learn
- NumPy
- Joblib

---

📊 Machine Learning Models
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Ensemble Model (Soft Voting)
